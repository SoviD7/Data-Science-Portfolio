# Data Science Portfolio

This repository contains my data science projects demonstrating data analysis, statistical testing, and machine learning.

## Projects

### 1. E-commerce Customer Behavior Analysis
Exploratory data analysis on retail transaction data to identify revenue trends, top products, and seasonal patterns.

### 2. A/B Testing for Website Conversion
Statistical analysis of an online experiment using a two-proportion Z-test to determine whether a new landing page improves conversion rate.

### 3. Customer Churn Prediction
Built machine learning models (Logistic Regression and Random Forest) to predict telecom customer churn and identify key drivers such as tenure and monthly charges.

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
